# Express Demo Server

## Description

This project is a simple **Node.js and Express.js server application** that demonstrates basic routing and HTTP server functionality. It includes multiple endpoints that return simple text responses when accessed through a browser or API client.

The project is useful for beginners who want to understand how to build a basic backend server using **Express** and **Node.js**.

---

## Features

* Simple Express server
* Multiple API routes
* Basic HTTP server using Node.js
* Beginner-friendly project structure

---

## Technologies Used

* Node.js
* Express.js

## Installation

1. Clone the repository

```
git clone https://github.com/your-username/your-repository-name.git
```

2. Navigate to the project folder

```
cd your-repository-name
```

3. Install dependencies

```
npm install
```

---

## Running the Application

Run the Express server:

```
node express-demo.js
```

Or run the Node.js HTTP server:

```
node server.js
```
